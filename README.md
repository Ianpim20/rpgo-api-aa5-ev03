\# RPGo API - AA5\_EV03



API REST desarrollada en Spring Boot para el proyecto formativo (Entrenador Personalizado / RP Go).



\## Módulos

\- Usuarios

\- Clientes

\- Rutinas



\## Tecnologías

\- Java

\- Spring Boot

\- Spring Web

\- Spring Data JPA

\- MySQL

\- OpenAPI/Swagger



\## Ejecución

1\. Configurar `application.properties`

2\. Ejecutar `RpgoApiApplication`

3\. Probar en Swagger:

&nbsp;  - http://localhost:8080/swagger-ui/index.html
